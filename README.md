# rdkit-tips-and-tricks
topic: presentation of molecules
In this topic, I am presenting the most used approaches to molecule presentation 

First: MolstoGrid 
presenting all molecules in one simple grid 
cases: 1- dataset visualisation 
2- mols comparing 

Second: MolsToFile 
presenting and saving each molecule in separate images 
cases :
1- paper writing 
2 - presentations
